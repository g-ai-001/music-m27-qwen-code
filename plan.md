# 项目规划

## 项目概述
- **项目名称**: music-m27-qwen-code
- **APP名称**: 本地音乐
- **包名**: app.music_m27_qwen_code
- **目标平台**: Android (API 26+)
- **核心功能**: 纯本地音乐播放器，支持扫描本地音频文件、管理歌单、显示歌词、高品质播放体验

## 开发规范

### 分支管理
- 主分支: `main`
- 所有开发基于 `main` 分支

### 版本号规范
- **MAJOR**: 重大不兼容更新
- **MINOR**: 新功能，向下兼容
- **PATCH**: Bug修复、代码重构，向下兼容

### 提交规范
- 采用 Conventional Commits 格式
- 使用简体中文提交信息

### 发布流程
1. 创建 Git Tag (格式: v*.*.*)
2. 推送 Tag 到 GitHub
3. GitHub Actions 自动构建并发布
